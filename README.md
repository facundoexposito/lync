# LYNC — Make Friends in Madrid

A Next.js 15 website for LYNC Events, helping international women connect and build community in Madrid.

## Tech Stack

- **Framework:** Next.js 15 (App Router, Turbopack)
- **Language:** TypeScript
- **Styling:** Tailwind CSS 4
- **Icons:** Lucide React
- **Deployment:** Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the site.

## Project Structure

```
src/
├── app/              # Next.js App Router pages
├── components/       # React components
├── data/            # Static data (quiz, events, testimonials)
└── lib/             # Utilities and types
```
