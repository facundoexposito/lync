1. How to Access the Studio

1
Go to lyncevents.com/studio in your browser.
2
Click "Sign in with Google" and use the Google account that was invited to the project.
3
You'll land on the Studio dashboard. On the left sidebar you'll see "Retreat" — this is where all retreat content lives.
Bookmark it! Save lyncevents.com/studio as a bookmark for quick access. You can use the Studio from any device with a browser.

2. Editing an Existing Retreat

1
Click "Retreat" in the left sidebar to see all retreats.
2
Click the retreat you want to edit. The editing panel opens on the right.
3
Make your changes — edit text, swap images, update pricing, etc.
4
Click the blue "Publish" button in the bottom-right corner. Changes go live on the website within seconds.
Auto-save drafts Every change is automatically saved as a draft. You won't lose work if you close your browser — just come back and hit Publish when ready.

3. Creating a New Retreat

1
Click the pencil icon (compose button) at the top of the document list, or click the "+" next to the Retreat type in the sidebar.
2
Fill in the three required fields:
  • Title — the retreat name (e.g., "Solstice")
  • Slug — click "Generate" to auto-create from the title
  • Hero Image — upload or select the main banner image
3
Add any optional sections you'd like — pricing, itinerary, daily schedule, inclusions, images, etc. Sections left empty simply won't appear on the website.
4
Click "Publish" when you're happy. The new retreat page goes live immediately.
Start minimal, add later You only need a title, slug, and hero image to publish. You can always come back to fill in pricing, itinerary, and other sections later.

4. Field-by-Field Guide

Core Info

FieldStatusWhat It Does
Title Required The retreat name displayed everywhere — page title, cards, navigation. Example: "Solstice"
Slug Required The URL path for the retreat. Click "Generate" to create it from the title. Creates URLs like lyncevents.com/retreats/solstice
Subtitle Optional A secondary tagline. Example: "A Journey Back to Your Heart"
Short Description Optional Brief description used in the intro section if no custom Intro Text is set
Location Optional Where the retreat takes place. Example: "Santa Teresa, Costa Rica"
Venue Optional Specific venue name. Example: "House of Shakti"
Dates Optional Display dates as you want visitors to see them. Example: "March 22–29, 2026"
Duration Optional Length of the retreat. Example: "7 nights"
Group Size Optional Expected group size. Example: "10 women"

Customizable Copy

FieldStatusWhat It Does
Intro Text Optional Custom intro paragraphs for the retreat. Separate paragraphs with a blank line. If empty, the Short Description is used instead.
Programme Title Optional Heading for the programme/itinerary section. If empty, defaults to "[Duration] of Becoming"
Programme Subtitle Optional Description text under the programme heading
Founder Story Title Optional Heading for the founder story section. If empty, defaults to "The Story Behind [Title]"
Daily Schedule Subtitle Optional Subtitle for the daily schedule section. If empty, defaults to "Every day is designed to be yours."

Pricing

FieldStatusWhat It Does
Pricing (array) Optional Add one or more room tiers. Each tier has a Type (e.g., "Shared Room"), a Price (e.g., "$2,250"), and a list of Perks. The layout adapts automatically — 1 tier centers, 2-3 tiers show as columns.
Deposit Note Optional Text shown below pricing tiers. Example: "50% deposit to reserve · Monthly payments available"

Itinerary

FieldStatusWhat It Does
Itinerary (array) Optional Day-by-day programme. Each entry has a Day number, an Emoji, a Title (e.g., "Arrival & Opening Circle"), and a list of Highlights.

Daily Schedule

FieldStatusWhat It Does
Daily Schedule (array) Optional A typical day's timetable. Each entry has a Time, a Title, a Description, and an Emoji.

What's Included

FieldStatusWhat It Does
Inclusions (array) Optional Things included in the retreat price. Each entry has an Emoji, Title, and Description.
Not Included (array) Optional Simple list of what's not included (e.g., "Flights", "Travel insurance"). Shown as a comma-separated list below inclusions.

Founder Story

FieldStatusWhat It Does
Founder Story Optional A personal narrative about the retreat's origin. Plain text — separate paragraphs with blank lines.
Founder Attribution Optional Attribution shown below the story. Example: "Rebecca, Founder of Lync Events Madrid"

Images

FieldStatusWhat It Does
Hero Image Required The large banner image at the top of the retreat page. Also used for social media previews.
Intro Image Optional Shown alongside the intro text. Displayed in portrait orientation on desktop.
Card Image Optional Used on retreat listing cards. If empty, the hero image is used instead.
Bento Images (array) Optional Photo grid. The layout adapts automatically: 1 image = full width, 2 = two columns, 3 = asymmetric grid, 4+ = full bento grid.
Slideshow Images (array) Optional Auto-rotating image carousel. Each image has an alt text field — describe the image for accessibility.

Links & Contact

FieldStatusWhat It Does
Booking URL Optional Link to the booking system. When set, a "Book Now" button appears in the pricing section and in a sticky bar at the bottom of the page.
Brochure Optional PDF file upload. When set, a "Download Brochure" button appears in the pricing section.
Contact Email Optional Email address shown in the pricing section as "Questions? Reach us at [email]".

5. Uploading Images

How to Upload

1
Click any image field — you'll see an upload area.
2
Drag & drop an image file onto the area, or click "Upload" to browse your files.
3
Once uploaded, click the image to open the editing view where you can set a hotspot.

Hotspot Cropping

After uploading, you can set a hotspot — a focal point that tells the website which part of the image to keep visible when it's cropped for different screen sizes. Drag the circle to the most important part of the image (usually a face or the center of the scene).

Why set a hotspot? Images display at different aspect ratios on different parts of the site (widescreen for the hero, portrait for bento grids, etc.). The hotspot ensures the important part of your image is always visible, no matter how it's cropped.

Recommended Image Sizes

ImageRecommended SizeAspect Ratio
Hero 2400 × 1350px or larger Wide landscape (16:9)
Intro 1200 × 1600px Portrait (3:4)
Card 1200 × 900px Landscape (4:3)
Bento 1200 × 900px or larger Mix of portrait & landscape
Slideshow 2400 × 1200px Wide landscape (2:1)
Image format tip JPG works best for photos. Keep file sizes under 5MB for fast loading — Sanity automatically optimises images for the web, but starting with a reasonable size helps.

6. Uploading a Brochure PDF

1
Scroll to the Brochure field in the retreat editor.
2
Drag & drop a PDF file onto the upload area, or click "Upload" to browse your files. Only .pdf files are accepted.
3
Once uploaded, a "Download Brochure" button will automatically appear on the retreat page (in the pricing section).

To replace a brochure, click the existing file and select "Clear field", then upload the new PDF.

7. Publishing Flow

→
Saved as Draft
→
Click Publish
→
Live on site

How It Works

  • Every edit you make is automatically saved as a draft. Nothing goes live until you publish.
  • The "Publish" button (blue, bottom-right corner) pushes your changes to the live website.
  • Changes typically appear on the site within a few seconds of publishing.
  • You can make multiple edits over time and publish them all at once.

Unpublishing

To take a retreat page offline, click the arrow next to Publish to reveal the dropdown menu, then select "Unpublish". The page will no longer be accessible on the website.

Draft vs Published If you see a yellow "Edited" badge next to a retreat, it means you have unpublished changes. Click "Publish" to push them live, or continue editing.

8. Section Visibility Rules

The retreat page is built to be flexible. Sections automatically show or hide based on whether you've filled in the content. If a section is empty, it simply doesn't appear on the page — no blank spaces, no broken layouts.

SectionShows When...If Empty...
Intro Image Intro Image is uploaded Intro text spans full width
Photo Bento Grid 1+ Bento Images added Section hidden
Image Slideshow 1+ Slideshow Images added Section hidden
Daily Schedule 1+ schedule entries added Section hidden
Programme / Itinerary 1+ itinerary days added Section hidden
Founder Story Founder Story text filled in Section hidden
What's Included 1+ inclusions added Section hidden
Not Included 1+ items added (within Inclusions) Subsection hidden
Pricing & CTA 1+ pricing tiers added Entire pricing section hidden
Book Now button Booking URL is set Button hidden
Download Brochure button Brochure PDF is uploaded Button hidden
Contact Email Contact Email is filled in Line hidden
Sticky Bottom Bar Booking URL is set & user scrolls past hero Bar hidden
The golden rule Fill in what you have, leave empty what you don't. The page always looks clean.

9. Tips & Troubleshooting

Paragraph Breaks

For text fields like Intro Text and Founder Story, separate paragraphs with a blank line (press Enter twice). Single line breaks are treated as one continuous paragraph.

Slugs

Always use the "Generate" button to create slugs from the title. If you need a custom URL, you can edit the slug manually — just use lowercase letters and hyphens only (e.g., spring-equinox-2027).

Don't change slugs after publishing Changing a retreat's slug changes its URL. If anyone has bookmarked or shared the old URL, it will stop working. Only change slugs before the first publish if possible.

Changes Not Showing?

  • Make sure you clicked "Publish" — drafts don't appear on the site.
  • Try a hard refresh in your browser: Cmd + Shift + R (Mac) or Ctrl + Shift + R (Windows).
  • If changes still aren't showing after a minute, reach out to your developer.

Replacing Images

Click on the existing image, then click "Clear field" to remove it. Then upload the new image. Remember to set the hotspot on the new image.

Pricing Perks

To add perks to a pricing tier, click into the tier and use the "Add item" button under Perks. Each perk is a single line of text (e.g., "Daily yoga sessions", "All meals included").
