3. Content Management (Sanity CMS)

Sanity is your content management system. It handles blog posts and retreats. You access it through the Studio, which is built right into your website.

Accessing the Studio

1
Go to lyncevents.com/studio in your browser
2
Sign in with the Google account that was invited to the project
3
You'll see two content types in the sidebar: Retreat and Blog Post
Bookmark it! Save lyncevents.com/studio for quick access. Works on any device with a browser.

Content Types

TypeWhat It ManagesURL Pattern
RetreatRetreat pages with pricing, itinerary, images, schedule/retreats/[name]
Blog PostBlog articles with sections, images, categories/blog/[title]

Publishing Flow

Edit content
→
Saved as Draft
→
Click Publish
→
Live on site
  • Every edit is auto-saved as a draft. Nothing goes live until you publish.
  • Click the blue "Publish" button (bottom-right) to push changes live.
  • Changes appear on the website within seconds.
  • No code deploy required — cache invalidation happens automatically.
Draft vs Published If you see a yellow "Edited" badge next to content, it means you have unpublished changes. Click "Publish" to push them live.

4. Managing Retreats

Creating a New Retreat

1
In the Studio, click "Retreat" in the sidebar, then the + button
2
Fill in the three required fields:
  • Title — the retreat name (e.g., "Solstice")
  • Slug — click "Generate" to create from the title
  • Hero Image — upload the main banner image
3
Add optional sections: pricing, itinerary, daily schedule, inclusions, images, founder story, etc.
4
Click "Publish" — the retreat page goes live immediately
Start minimal, add later You only need a title, slug, and hero image to publish. Fill in pricing, itinerary, and other sections whenever you're ready.

5. Blog & Admin Panel (AI-Powered)

The admin panel at lyncevents.com/admin lets you generate blog posts using AI. You provide a brief, Claude AI writes the post matching your brand voice, and you publish it directly to Sanity.

Accessing the Admin Panel

1
Go to lyncevents.com/admin
2
Enter the admin password
3
Click "New blog post" to start

Creating a Blog Post

1
Fill in the brief:
  • Working title — rough title idea (AI will polish it)
  • Category — Nightlife, Meetups, or Wellness
  • Note from the founder — your angle, key message, what you want to say
  • Key questions/topics (optional) — these become section headings
  • Cover image — upload a photo (jpg, png, or webp)
2
Click "Generate draft" — AI writes the full post in ~10 seconds, matching your brand voice.
3
Review the preview. You can edit the title, excerpt, and all section text before publishing.
4
Click "Publish post" — the image uploads to Sanity, the blog document is created, and the post goes live within seconds.

Blog Post Structure

Every AI-generated post follows this structure:

  • 3-5 sections total, ~400-550 words
  • First section: no heading (intro), 2 short paragraphs
  • Other sections: evocative heading + 1-2 paragraphs
  • Ends with a soft nudge to LYNC events
  • No bullet lists, no emoji in body, no fake quotes
Brand voice Warm, confident, honest. Like a slightly older friend giving advice over coffee. Madrid-specific details welcome. Never corporate, never hype-y.

6. Blog via Sanity Studio

You can also create or edit blog posts directly in the Sanity Studio. This is useful for editing posts after they've been published, or creating posts without the AI flow.

Blog Post Fields

FieldStatusWhat It Does
TitleRequiredPost title on cards and article page
SlugRequiredURL path — click "Generate" from title
DateRequiredPublication date shown on the article
CategoryRequiredNightlife, Meetups, or Wellness
ExcerptRequiredShort summary for cards and SEO (~180 chars)

7. Events (Acuity Scheduling)

Events on the website are pulled automatically from your Acuity Scheduling account. Just create events in Acuity and they appear on the site.

How It Works

1
You create events/classes in Acuity Scheduling as usual
2
The website fetches upcoming events every 5 minutes
3
Events are auto-categorized based on keywords in their title or description
4
Visitors can filter by category and click to book directly through Acuity

Auto-Categorization

CategoryKeywords Detected
Wellnessyoga, pilates, meditation, breathwork, sound bath, spa, stretch, mindful
Adventurerun, hike, climb, outdoor, kayak, surf, bike, walk, tour, explore
Creativecraft, paint, art, pottery, workshop, cook, bake, flower, vision board
Nightlifeclub, party, rooftop, cocktail, bar crawl, night out, DJ
Socialbrunch, dinner, lunch, coffee, picnic, game night, meetup
Tip Include descriptive keywords in your Acuity event titles for correct categorization. E.g., "Sunset Yoga in Retiro" will be tagged as Wellness. If no keyword matches, the event defaults to "Social".

If an event doesn't have a photo in Acuity, the website uses a category-specific fallback image from your brand photos.

11. Troubleshooting

Blog post not appearing after publishing

  • Make sure you clicked "Publish" in the Studio or Admin Panel.
  • Hard refresh: Cmd+Shift+R (Mac) or Ctrl+Shift+R (Windows)
  • If still not showing after a minute, the Sanity webhook may need checking. The webhook URL should be: lyncevents.com/api/webhooks/sanity?secret=YOUR_SECRET

Events not showing on the website

  • Events refresh every 5 minutes. Wait a moment if you just created one in Acuity.
  • Make sure the event is published with upcoming dates in Acuity.
  • Verify ACUITY_USER_ID and ACUITY_API_KEY are set in Vercel.

Quiz submissions not in Airtable

  • Check AIRTABLE_PAT and AIRTABLE_BASE_ID in Vercel.
  • Make sure the table name matches AIRTABLE_TABLE_NAME (default: "Leads").

Admin panel login fails

  • ADMIN_PASSWORD must be set in Vercel and match exactly (case-sensitive).

AI blog generation fails

  • Check ANTHROPIC_API_KEY is set and valid at console.anthropic.com.

Replacing images in the Studio

Click the existing image → "Clear field" → upload the new one. Set a hotspot (drag the circle to the important part of the image).
